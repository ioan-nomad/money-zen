# MoneyZen v1.0.0 - Portable Edition

## How to Run:
1. Double-click MoneyZen.exe
2. Windows Defender might warn (it's safe, just not signed)
3. Click "More info"  "Run anyway"

## First Run:
- Database will be created at: %LOCALAPPDATA%\io.moneyzen.app\
- All data is stored locally on your PC
- No internet required

## Features:
- 12 N-OMAD accounts pre-configured
- 32 expense/income categories
- Transaction management
- Excel import/export
- PDF reports
- Tag system
- Backup/restore
